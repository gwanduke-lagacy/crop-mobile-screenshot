# Crop Mobile Screenshot

안드로이드 기기에서 스크린샷하면 상단 상태바와 하단 소프트키가 포함되는데 이를 제거해주는 프로그램

## 현재 스펙

- 안드로이드 GalaxyS10+ 기준

## 사용방법

- images/source 폴더에 변환을 원하는 스크린샷들을 복사

```bash
$ yarn start
# => Yeah!
```
